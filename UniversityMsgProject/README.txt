----------------------------------- Application d'annonces -----------------------------------

L'application ne fonctionne que si la base de donnée est initialisée avant de lancer les exécutables.

Afin d'initialiser la base de donnée, il faut executer sur le serveur Wamp le script SQL "bddMySQL.sql"
contenu dans le dossier "ProjetS5\sqlBDD\".

On peux ensuite lancer les exécutables.
(Il faut lancer le serveur avant le(s) client(s) pour que l'application fonctionne).

Pour exécuter le client, il faut exécuter le fichier "Serveur.jar"
Pour exécuter le client, il faut exécuter le fichier "Client.jar"
